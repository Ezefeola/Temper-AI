# Code Conventions

All conventions below apply to every .NET project generated by TemperAI.

---

## Absolute Rules — Never Broken

| Rule | Bad | Good |
|---|---|---|
| **No primary constructors** | `public class Product(string name)` | `public class Product { public Product(string name) { ... } }` |
| **No `=>` on methods** | `public string GetName() => Name;` | `public string GetName() { return Name; }` |
| **No DataAnnotations** | `[MaxLength(100)] public string Name` | Configured in `IEntityTypeConfiguration<T>` |
| **No `.Update()` in EF Core** | `_dbContext.Products.Update(product)` | Modify tracked entity, call `SaveChangesAsync` |
| **No `async void`** | `public async void DoWork()` | `public async Task DoWork()` |
| **No `.Result` / `.Wait()`** | `var x = task.Result` | `var x = await task` |
| **No `using static`** | `using static System.Console;` | `using System;` then `Console.WriteLine()` |
| **No named usings** | `using TodoTask = Domain.Task;` | `using Domain;` then `Task` |
| **No global usings** | `global using Microsoft.AspNetCore.Mvc;` | Per-file `using Microsoft.AspNetCore.Mvc;` |
| **No `var`** | `var product = ...` | `Product product = ...` |
| **No fully qualified types** | `public Projects.Enums.Status Status` | `using Projects.Enums;` then `public Status Status` |
| **No `throw` for validations** | `throw new ArgumentException(...)` | Return `(List<string> Errors, Entity? Entity)` |
| **No `nvarchar(max)` / `varchar(max)`** | `.HasColumnType("nvarchar(max)")` | `.HasMaxLength(Entity.Rules.NAME_MAX_LENGTH)` |
| **No lazy loading** | `.UseLazyLoadingProxies()` | Explicit `.Include()` |
| **No hardcoded numbers in validators** | `.MaximumLength(100)` | `.MaximumLength(Product.Rules.NAME_MAX_LENGTH)` |
| **No unnecessary line breaks** | Multi-line short assignments | Single line if it fits |
| **No Spanish in code** | `Estado.Activo`, `EsRequerido` | `Status.Active`, `IsRequired` |

## Always Required

| Rule | Example |
|---|---|
| `sealed` on non-inherited classes | `public sealed class Product` |
| `CancellationToken` on public async | `Task<Product?> GetByIdAsync(Guid id, CancellationToken ct = default)` |
| Variable names match type | `SaveResult saveResult`, `Product product` |
| `varchar` for ASCII, `nvarchar` for Unicode | `.HasColumnType("varchar")` for names |
| One `IEntityTypeConfiguration<T>` per entity | `ProductConfiguration.cs` |
| `GetByIdAsync` with tracking | For modifications |
| `GetByIdAsNoTrackingAsync` without tracking | For read-only queries |
| DTOs as `sealed record` with `Dto` suffix | `CreateProductRequestDto` |
| Use cases without `UseCase` suffix | `CreateProduct`, not `CreateProductUseCase` |
| Factory methods return `(Errors, Entity?)` | `public static (List<string> Errors, Product? Product) Create(...)` |
| Update methods return `(Errors, bool Updated)` | `public (List<string> Errors, bool Updated) UpdateName(...)` |
| Nested `Rules` class | `public class Rules { public const int NAME_MAX_LENGTH = 100; }` |
| `UpdatedAt` on every update | Set explicitly in update methods |
| `[FromServices]` per endpoint | No general constructor in controllers |
| `result.ToActionResult()` | Never build responses manually |
| `ProblemDetails` for errors | With `errors` extension field |
| Code in English | Except user-facing error messages |

## Naming Conventions

| Element | Convention | Example |
|---|---|---|
| Classes | `PascalCase` | `Product`, `CreateProductHandler` |
| Interfaces | `I` + `PascalCase` | `IProductRepository`, `ICreateProduct` |
| Methods | `PascalCase` + `Async` suffix | `GetByIdAsync`, `CreateAsync` |
| Properties | `PascalCase` | `Name`, `CreatedAt` |
| Fields (private) | `_camelCase` | `_unitOfWork`, `_logger` |
| Local variables | `camelCase` — match type | `SaveResult saveResult` |
| Parameters | `camelCase` | `cancellationToken` |
| Constants | `PascalCase` in `Rules` | `Rules.NAME_MAX_LENGTH` |
| DTOs | `PascalCase` + `Dto` suffix | `CreateProductRequestDto` |
| Use cases | `PascalCase` — no suffix | `CreateProduct` |
| Events | `PascalCase` + `Event` | `ProductCreatedEvent` |
| Enums | `PascalCase` | `ProductStatus` |
| Entity folders | **Plural** | `Domain/Products/Product.cs` |
| EF configs | Entity + `Configuration` | `ProductConfiguration` |

## Entity Pattern

```csharp
public sealed class Product : Entity<Guid>
{
    public class Rules
    {
        public const int NAME_MAX_LENGTH = 100;
        public const int DESCRIPTION_MAX_LENGTH = 500;
        public const decimal MIN_PRICE = 0;
    }

    public string Name { get; private set; } = string.Empty;
    public string Description { get; private set; } = string.Empty;
    public decimal Price { get; private set; }
    public ProductStatus Status { get; private set; }
    public DateTime CreatedAt { get; private set; }
    public DateTime? UpdatedAt { get; private set; }

    private Product() { }

    public static (List<string> Errors, Product? Product) Create(
        string name,
        string description,
        decimal price)
    {
        List<string> errors = [];

        if (string.IsNullOrWhiteSpace(name))
            errors.Add("Name is required");
        else if (name.Length > Rules.NAME_MAX_LENGTH)
            errors.Add($"Name cannot exceed {Rules.NAME_MAX_LENGTH} characters");

        if (string.IsNullOrWhiteSpace(description))
            errors.Add("Description is required");
        else if (description.Length > Rules.DESCRIPTION_MAX_LENGTH)
            errors.Add($"Description cannot exceed {Rules.DESCRIPTION_MAX_LENGTH} characters");

        if (price <= Rules.MIN_PRICE)
            errors.Add("Price must be greater than zero");

        if (errors.Count > 0)
            return (errors, null);

        Product product = new()
        {
            Id = Guid.NewGuid(),
            Name = name,
            Description = description,
            Price = price,
            Status = ProductStatus.Active,
            CreatedAt = DateTime.UtcNow
        };

        return ([], product);
    }

    public (List<string> Errors, bool Updated) UpdateName(string newName)
    {
        List<string> errors = [];

        if (string.IsNullOrWhiteSpace(newName))
            errors.Add("Name is required");
        else if (newName.Length > Rules.NAME_MAX_LENGTH)
            errors.Add($"Name cannot exceed {Rules.NAME_MAX_LENGTH} characters");

        if (errors.Count > 0)
            return (errors, false);

        if (Name == newName)
            return ([], false);

        Name = newName;
        UpdatedAt = DateTime.UtcNow;
        return ([], true);
    }
}
```

## Result Pattern

```csharp
public sealed class Result<TResponse>
{
    public bool IsSuccess { get; private set; }
    public HttpStatusCode HttpStatusCode { get; private set; }
    public string Description { get; private set; } = string.Empty;
    public List<string> Errors { get; private set; } = [];
    public TResponse? Payload { get; private set; }

    public static Result<TResponse> Success(HttpStatusCode code) { ... }
    public static Result<TResponse> Failure(HttpStatusCode code) { ... }
    public Result<TResponse> WithDescription(string desc) { ... }
    public Result<TResponse> WithErrors(List<string> errors) { ... }
    public Result<TResponse> WithPayload(TResponse payload) { ... }
}
```

## DTO Pattern

```csharp
public sealed record CreateProductRequestDto
{
    public string Name { get; init; } = string.Empty;
    public string Description { get; init; } = string.Empty;
    public decimal Price { get; init; }
}

public sealed record CreateProductResponseDto
{
    public Guid Id { get; init; }
    public string Name { get; init; } = string.Empty;
    public decimal Price { get; init; }
    public string Status { get; init; } = string.Empty;
}
```

## Controller Pattern

```csharp
[ApiController]
[Route("api/[controller]")]
public class ProductsController : ControllerBase
{
    [HttpPost]
    [ProducesResponseType(typeof(CreateProductResponseDto), StatusCodes.Status201Created)]
    [ProducesResponseType(typeof(ProblemDetails), StatusCodes.Status400BadRequest)]
    [ProducesResponseType(typeof(ProblemDetails), StatusCodes.Status409Conflict)]
    public async Task<IActionResult> Create(
        [FromBody] CreateProductRequestDto request,
        [FromServices] ICreateProduct createProduct,
        CancellationToken cancellationToken)
    {
        Result<CreateProductResponseDto> result =
            await createProduct.ExecuteAsync(request, cancellationToken);

        return result.ToActionResult();
    }
}
```

## API Documentation — Scalar

Always use Scalar for API documentation. Never Swagger/Swashbuckle.

```csharp
builder.Services.AddOpenApi();

if (app.Environment.IsDevelopment())
{
    app.MapOpenApi();
    app.MapScalarApiReference(options =>
    {
        options.WithTitle("ProjectName API");
        options.WithTheme(ScalarTheme.Kepler);
    });
}
```

- OpenAPI JSON: `/openapi/v1.json`
- Scalar UI: `/scalar/v1`

## API and Frontend Separation

**Never** put the API and Frontend in the same solution.

```
TodoManagerApi/         ← TodoManagerApi.sln
├── src/
│   └── TodoManagerApi/
└── tests/

TodoManagerFront/       ← TodoManagerFront.sln
├── src/
│   └── TodoManagerFront/
└── tests/
```

- Separate `.sln` files
- Sibling directories, never nested
- Frontend communicates with API only via HTTP
- No shared projects, no project references
